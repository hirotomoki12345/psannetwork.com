自動クリックコードの例

このコードは、JavaScriptで実装された自動クリック機能のデモンストレーションです。クラス名が「auto-clicker-target」の要素がクリックされると、自動クリックがオンとオフが切り替わります。自動クリックは指定された遅延時間で要素をクリックします。

以下のボタンをクリックして、コードをコピーできます。


var DELAY = 1;

function autoClick(element) {
  element.click();
  setTimeout(function() {
    autoClick(element);
  }, DELAY);
}

document.body.addEventListener("click", function(e) {
  if (e.target.classList.contains("auto-clicker-target")) {
    e.target.classList.remove("auto-clicker-target");
  } else {
    e.target.classList.add("auto-clicker-target");
  }
  autoClick(e.target);
});
